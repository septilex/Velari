# Velari

Velari is a creative drawing space made by me and Harshitha for fun, creativity, relaxation, and imagination ✨

We built this project because we wanted a calm and beautiful place where anyone can freely draw, think, experiment, and express ideas without pressure.

This project is open to everyone and all age groups.

Whether you are:
- sketching random ideas
- relaxing after a long day
- creating art
- testing creativity
- or just having fun

Velari is for you 💜

We are still learning and improving the project every day, so your support and encouragement truly mean a lot to us.

Thank you for visiting and supporting Velari 🚀

— Prajju & Harshitha
